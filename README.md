# Chiron Mediplus Pharmacy

Static website for Chiron Mediplus Pharmacy built with HTML, CSS, and JavaScript.

## Project Structure
- `Chiron_Mediplus_Phamarcy/` Main site
- `Chiron_Mediplus_Extras/` Extra home templates and unused pages

## Run Locally
Open `Chiron_Mediplus_Phamarcy/index.html` in your browser or use a local server like VS Code Live Server.

## Key Features
- Upload prescription flow via WhatsApp
- Pharmacy‑specific service request forms
- Delivery policy and privacy/terms content

## Notes
This project uses third‑party libraries included in `Chiron_Mediplus_Phamarcy/assets/`.
